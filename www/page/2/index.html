Linux ip 端口选择

网络通信中有2种角色，client与server，分别执行不同的操作：

server
listen
client
connect

server的端口相对固定，而且是知名端口，或者是配置文件中配置的。

对于client连接远程客户端的时候，需要选取本地的端口和ip来建立连接。
