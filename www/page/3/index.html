题未定

爸妈来杭州了，今天我去上班，回来发现，他们俩自个儿逛到余杭镇，买了好多菜和家居用品，房子也收拾的很干净了，顿时感觉到了家的味道。

我自己也感觉到了一点点不知不觉的幸福————关于生活的、关于家庭的。

坐在马桶上，看着打扫干净的卫生间，擦亮的门把手。不自觉得在想：为什么总是那么忙呢？难道就是所谓的奋斗吗？

也许，我们也可以慢下来，放松些，脑子里不仅仅无时无刻不想着要高效、快速，而是抽出20%的时间，放松放松，做点无论是什么的事情，没有意义，没有价值，没有效率的时间，只为了让自己大脑放松，心情放空，感受呼吸，感受外在。

想想也许为何不可呢？

IP 地址/路由的 scope

ip route/ip adress/ifconfig 命令的输出，会发现有一个scope选项。

对于绑定了地址的网口，一般是 : Link 。对于 lo接口，是： Host 。

site 主要用在 ipv6中，ipv4网络没有这个。

global 只出现在路由中，也是路由的默认 Scope，表示全局有效。

Scope | Description

global | valid everywhere (GLOBAL NET)

site | valid only within this site (IPv6)

link | valid only on this device (LAN)

host | valid only inside this host (machine)
